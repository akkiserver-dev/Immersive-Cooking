# Immersive Cooking
**An Immersive Engineering addon adds multiblock for Farmer's Delight/Let's Do Series Automation**

## Features
### Multiblocks
- Industrial Cookpot
  - Automates Farmer's Delight / Farm & Charms Cooking Pot
- Grill Oven
  - Multithreaded Smoker
